## Проблема

**HR-менеджеры тратят до 40% рабочего времени на рутинный первичный скрининг резюме.**

- 📄 Сотни резюме на одну вакансию
- ⏰ Дни на телефонные интервью с неподходящими кандидатами
- 💰 Потеря денег на интервью с кандидатами, которые не проходят дальше
- 😫 Выгорание HR от монотонной работы

---

## Решение

**CVortex автоматизирует первичный отбор кандидатов с помощью AI.**

Платформа самостоятельно:
- ✅ Анализирует резюме и оценивает соответствие требованиям
- ✅ Проводит первичное интервью через Telegram-бота
- ✅ Ранжирует кандидатов по релевантности
- ✅ Передаёт HR только тех, кто прошёл отбор

---

## Результат

| Метрика | До | После |
|---------|-----|-------|
| Время на первичный скрининг | 3-5 дней | 30 минут |
| Телефонные интервью с неподходящими | 70% | 15% |
| Нагрузка на HR | Высокая | Минимальная |
| Скорость закрытия вакансии | 3-4 недели | 1-2 недели |

## Как это работает

```
┌─────────────────────────────────────────────────────────────┐
│                      HR-менеджер                            │
│  Создаёт вакансию → Указывает требования → Получает        │
│  готовых кандидатов, прошедших AI-отбор                     │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      Кандидат                               │
│  Регистрируется в Telegram → Загружает резюме → Проходит   │
│  автоматическое интервью → Получает результат              │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                         AI                                  │
│  Анализирует резюме → Оценивает ответы → Фильтрует         │
│  неподходящих → Передаёт лучших HR                          │
└─────────────────────────────────────────────────────────────┘
```

### Процесс за 4 шага

1. **HR создаёт вакансию** — описывает требования к кандидату и вопросы для интервью
2. **Кандидат регистрируется** — через Telegram-бота загружает резюме
3. **AI проводит скрининг** — оценивает резюме и проводит интервью
4. **HR получает отчёт** — только кандидаты, прошедшие порог 75%

## Ключевые возможности

### 🤖 AI-скрининг резюме

Yandex GPT автоматически извлекает навыки, опыт и образование из PDF-резюме и сравнивает с требованиями вакансии. Кандидаты с оценкой ниже 75% отсеиваются.

### 💬 Автоматические интервью

Telegram-бот проводит структурированные интервью по вашим вопросам. Каждый ответ оценивается AI индивидуально — без участия HR.

### 📊 Единая панель управления

Веб-интерфейс для HR-менеджеров:
- Создание и управление вакансиями
- Просмотр всех кандидатов с оценками
- Детализация по каждому ответу
- Экспорт данных

### 🔒 Безопасность

- Авторизация через Yandex OAuth 2.0
- JWT-токены для API
- Резюме хранятся в Yandex Object Storage
- Персональные данные защищены
